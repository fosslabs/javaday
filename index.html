Java Day Казань 2012 Вторая ежегодная конференция Java разработчиков

3 марта 2012 года в Казани состоится специализированная конференция Java разработчиков Java Day Казань 2012.

В программе конференции интересные и познавательные доклады от инженеров компании Oracle из Москвы и Санкт-Петербурга, а также доклады от представителей казанских ИТ-компаний, активно пользующихся Java технологиями, и сообщества Java разработчиков.

Участвуя в мероприятии, вы узнаете о планах компании Oracle по развитию технологий JavaSE, JVM, JavaEE, JavaME, Embedded Java, JavaFX, об улучшении прозводительности Java и, конечно, сможете пообщаться с коллегами.

Участие в конференции бесплатное, требуется лишь предварительная регистрация.

Java Day Казань 2012 пройдет в ГАУ «Технопарк в сфере высоких технологий «ИТ-парк», по адресу ул. Петербургская, 52.

До встречи на Java Day Казань 2012!
